System Parameters

These parameters control global behavior of the EV-DMS platform. (Demo: hiện tại mới là dữ liệu tĩnh, sau bạn có thể cho chỉnh & lưu vào DB.)

Key Value Description
Max Users per Dealer 50 Maximum number of user accounts allowed under a single dealer.
Default Currency VND Default currency for prices and reports.
Session Timeout (minutes) 30 User will be logged out automatically after this idle time.
